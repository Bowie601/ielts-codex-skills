# IELTS Codex Skills

Public collection of IELTS-related Codex skills.

## Skills

- `ielts-task1-paraphrase-coach`: Interactive coach for IELTS Academic Writing Task 1 prompt paraphrasing.

## Install

Install the current skill with:

```bash
python3 ~/.codex/skills/.system/skill-installer/scripts/install-skill-from-github.py \
  --repo Bowie601/ielts-codex-skills \
  --path skills/ielts-task1-paraphrase-coach
```

Restart Codex after installation.
